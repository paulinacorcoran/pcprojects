Welcome to my personal site!

Purpose of this Site

Hi! I’m Paulina and I am a student at the University of Miami. This semester, I took CIM 111, an introductory coding course using html, css, and jQuery to develop websites. This website serves as a personal site to feature information about myself, my education, and my resume. Aside from being a personal website, it is also my final project for the course. The task is to create a functional web page using all of the skills learned within the CIM 111 course. My site uses elements of html, css, and jQuery in order to put this together. Navigate between the pages using the buttons at the top of the screen! To learn more about me, click on “About Me”. To learn more about my experience at the University of Miami, click on “Education”. Lastly, to view my resume, click “Resume”. If at any point you want to return to this webpage, click on “Index”. Thank you for viewing!
